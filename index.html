About this solution

This site was created with Start Bootstrap, Grayscale Theme. The theme is open source, and you can use it for any purpose, personal or commercial.

The solution for fitting the text within a container was done using simple javascript, found in fitText.js.

Suggestions of Improvements

  • The fitText function should listen for size changes to the parent element.
  • The fitText function should listen for text changes for the element.
  • Align text to the middle of the parent container in all situations.
  • Slider max value should not exceed the viewport width.
  • Add unit tests for fitText function.
  • Refactor code utilising ES6. (add Babel or other lybraries to convert javascript code before minifying).
